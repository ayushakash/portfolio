EXPRESS PARCEL

Project Overview

Express Parcel is a React courier delivery application which takes address of sender and receiver and calculates real-time road distance and delivery time and sends confirmation email to sender and receiver with traccking number.Here Distance matri APIs are used to calculate distance and time.

Key Features

  • 1. Real-time distance and time is displayed
  • 2. User get unique tracking number
  • 4. Confirmation mail is sent to user once order is placed.

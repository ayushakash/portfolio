CHARDEEVARI

Project Overview

Chardeevari is a E-Commerce web application for construction material.It aims at providing fastest delivery in town by giving order to the nearest store from shipping address.

Key Features

  • 1. Login and Signup connected to database.
  • 2. Google signup Integrated.
  • 3. Order is received by the nearest store to the shipping address.
  • 4. Confirmation mail is sent to user once order is placed.
